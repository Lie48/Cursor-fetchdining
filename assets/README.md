# Assets Directory

## Required Images

Please add the following image files to this directory:

### Logo Images
- `uindy-logo-night.png` - UIndy logo for dark/night mode
- `uindy-logo.png` - UIndy logo for light/day mode

### Dining Location Images
- `starbucks.jpg` - Starbucks location image
- `perk.jpg` - Perk location image
- `asada.jpg` - Asada location image
- `hanna.jpg` - Hanna Garden location image
- `streets.jpg` - Streets location image
- `sushi.jpg` - Sushi Boss location image

### App Icons (for production)
- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon

## Image Specifications

**Dining Location Images:**
- Recommended size: 500x360px or larger
- Format: JPG or PNG
- Aspect ratio: ~5:3

**Logo Images:**
- Recommended size: 300x160px or larger
- Format: PNG with transparency
- Background: Transparent for night mode, white for day mode
